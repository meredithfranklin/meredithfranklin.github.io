R Tutorials

Through teaching statistics courses at USC over the past few years, I have amassed a significant amount of instructional R code, from introductory statistics to advanced spatial modeling.
I am in the process of organizing all of this code into a set of tutorials and will be gradually publishing it through this site. You can see the table of contents on the left sidebar. Some of the links are empty – this is a work in progress!

I’ve written most of this site using Markdown and R Markdown (with RStudio), and am hosting it through GitHub pages.
